For Users

Wiki

The wiki contains plenty of information for new users.

If you're just setting up a new server or you're curious about what you can configure, feel free to check it out.

Resources Forum

The forum is an ideal place to share resources and find resources that other users have shared.

Feel free to take a look at the specific category we created for this purpose.
 

For Developers

Contributor Documentation

If you're interested in contributing to the project, there's a section on the wiki just for you!

We look forward to seeing your contributions! If you need any more help, feel free to ask us on Discord.

Glowkit JavaDocs

If you're a plugin developer, you can have a look at the JavaDocs for our fork of the Bukkit API.

Glowkit is based on PaperSpigot's Bukkit API fork, and it supports everything provided by that.

Glowstone JavaDocs

If you're interested in contributing or working with Glowstone internals, take a look at the JavaDocs for it.

Please don't be afraid to contact us if you're not sure how you can contribute!

Contributor License Agreement

All contributions to Glowstone are required to meet our Contributor License Agreement going forward.

Please make sure you read it, as you'll be asked to sign it when you open your next PR!

Site API Docs

Our website exposes a relatively simple REST API for everyone to make use of.

Please contact us directly if you need access to any of the data-writing APIs!
